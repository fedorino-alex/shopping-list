# Supermarket Sections

This is not a grocery-only store. The supermarket has the following departments:

| Отдел | Примеры товаров |
|---|---|
| Изделия из ткани | подушки, одеяла, простыни, наволочки, комплекты постельного белья, скатерти |
| Посуда | кастрюли, сковородки, тарелки, столовые приборы, кружки, чашки, контейнеры |
| Электротехника | электротовары, бытовая техника |
| Отдых на природе | товары для гриля, садовая мебель |
| Товары для животных | корма, наполнитель для лотка |
| Детские игрушки | лего, конструкторы, плюшевые игрушки |
| Канцелярские товары | ручки, карандаши, фломастеры, тетради, календари |
| Спортивные товары | спортинвентарь, одежда для спорта |
| Бытовая химия | стиральные порошки, мыло, средство для посудомойки, средство для посуды, моющие средства для кухни и унитаза, губки, тёрки, швабры |
| Одежда | мужская, женская, детская |
| Бакалея | орехи, сухофрукты, конфеты, печенье, вафли, пряники |
| Овощи и фрукты | свежие овощи, фрукты, готовые овощные салаты |
| Мясной отдел | свинина, говядина, курица, индейка, колбасы на развес, колбасы целые, мясо в нарезке |
| Крупы и макароны | гречка, рис, макароны, булгур, чечевица |
| Рыбный отдел | свежая рыба, солёная рыба, лосось |
| Сырный отдел | сыры всех видов в упаковках и на развес |
| Приправы | специи, приправы |
| Завтраки | хлопья, мюсли, каши быстрого приготовления |

| Алкоголь | пиво, вина, крепкие алкогольные напитки |
| Напитки | соки, вода, минеральная вода, газированная вода |

## Usage

Reference this file when updating the system prompt in `src/extractor.ts` to ensure
the LLM correctly identifies non-grocery items (textiles, kitchenware, electronics,
toys, etc.) as valid shopping list entries and doesn't strip or misinterpret them.
